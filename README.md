# Projeto Tela de Login

Este projeto foi desenvolvido para a atividade avaliativa de Git e GitHub.
Ele contém:

- Página de login
- Página de dashboard
- Página de cadastro
- Validação com JavaScript
- Versionamento com Git
- Uso de branches: main, develop, feature, hotfix e release

## Tecnologias usadas

- HTML
- CSS
- JavaScript
- Git e GitHub
